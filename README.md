# Restaurant Finder micro-service

### Instructions
1. Compiling the project source

   `sbt compile`

2. Compiling the project tests course

   `sbt test:compile`

3. Run project

   `sbt run`

4. Run tests

   `sbt test`


Compile and run backend app

```scala
sbt compile run
```

Compile and run backend app tests

```scala
sbt test:compile test
```
